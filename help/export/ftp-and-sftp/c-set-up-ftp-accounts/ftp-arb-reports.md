# Rapports distribués sur le Report Builder

Utilisez le module complémentaire Adobe Report Builder pour Microsoft Excel pour générer des demandes personnalisées d’après les données d’analyse Web, que vous pouvez ensuite insérer dans les classeurs Excel.

Les requêtes peuvent référencer des cellules de manière dynamique dans votre feuille de calcul et vous pouvez mettre à jour et personnaliser la présentation des données. Pour les protocoles FTP ou SFTP, vous devez vous assurer que l’hôte est disponible en externe. Pour plus d’informations sur les rapports distribués [!UICONTROL Report Builder], consultez l’aide de [Report Builder](/help/analyze/report-builder/rb-overview.md).

Pour plus d’informations sur les limites FTP ou SFTP et la conservation des données, voir [Limites et conservation des données FTP et SFTP](/help/export/ftp-and-sftp/ftp-limits.md).
