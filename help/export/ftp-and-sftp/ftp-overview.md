# Utilisation de FTP et SFTP avec Adobe CX Enterprise

Découvrez comment utiliser le protocole FTP et SFTP pour transférer des fichiers entre ordinateurs et serveurs.

Vous pouvez utiliser le protocole FTP avec les solutions et les produits Adobe des manières suivantes :

* Chargement des jeux de données de classification volumineux vers Analytics.
* Chargez les attributs du client vers CX Enterprise pour les utiliser dans les solutions.
* Utilisez Analytics pour créer et gérer des sources de données basées sur un FTP qui importent des données hors ligne ou historiques dans l’entreprise CX à l’aide du transfert de fichiers par FTP

Ces articles ne vous guident pas nécessairement tout au long des étapes nécessaires à l’utilisation du protocole FTP avec chaque produit. Il recommande plutôt les bonnes pratiques à suivre pour utiliser le protocole FTP avec Adobe.
