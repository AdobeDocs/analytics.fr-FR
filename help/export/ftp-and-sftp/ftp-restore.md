# Restaurer les comptes et données FTP ou SFTP supprimés

Les données supprimées d’un compte FTP ou SFTP ne peuvent pas être restaurées par Adobe.

Si un compte FTP ou SFTP que vous aviez l’intention d’utiliser a été désactivé, contactez l’équipe chargée de votre compte Adobe, qui pourra discuter des autres options avec vous.
