# Mise en oeuvre du Activity Map

Activity Map peut être implémenté avec une case à cocher dans l’extension Adobe Experience Platform Launch pour Adobe Analytics, sous [Suivi de liens](https://experienceleague.adobe.com/docs/launch/using/extensions-ref/adobe-extension/analytics-extension/overview.html?lang=en#link-tracking).
