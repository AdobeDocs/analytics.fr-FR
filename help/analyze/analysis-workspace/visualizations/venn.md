# Venn

Le [!UICONTROL Venn] la visualisation vous permet de faire glisser jusqu’à trois segments (depuis [!UICONTROL Composants]) et une mesure pour créer une [!UICONTROL Venn] diagramme.

![](assets/venn.png)

Vous pouvez ensuite pointer sur les segments afin d’afficher davantage d’informations sur les pourcentages, etc.

![](assets/venn_hover.png)

## Tourner [!UICONTROL Venn] diagramme en [!UICONTROL Structure libre] table

Pour transformer le diagramme de Venn en tableau à structure libre, cliquez sur le point coloré en regard de l’en-tête de **[!UICONTROL Venn]** et sélectionnez **[!UICONTROL Gérer les sources de données]** > **[!UICONTROL Sources de données disponibles]** > **[!UICONTROL Tableau à structure libre]**.

## Normaliser [!UICONTROL Venn] diagramme

Pour normaliser la variable [!UICONTROL Venn] (sa taille), cliquez sur l’icône Paramètres de visualisation et sélectionnez **[!UICONTROL Normalisation]**.

![](assets/normalization.png)
