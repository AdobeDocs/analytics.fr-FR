# Packs de contenu Adobe Analytics pour Power BI

{{legacy-arb}}

Les packs de contenu Adobe Analytics fournissent des tableaux de bord Power BI préconfigurés et un ensemble de rapports Power BI qui fournissent des informations sur le trafic sur votre site et les dimensions utilisateur. Vous pouvez utiliser le tableau de bord et les rapports fournis ou les personnaliser pour mettre en évidence les informations qui vous intéressent le plus. Les données sont automatiquement actualisées une fois par jour.

Trois packs de contenu sont disponibles :

* Adobe Analytics
* Adobe Analytics Mobile App Analytics
* Analyse du trafic Adobe Analytics

Pour plus d’informations, voir [Pack de contenu Adobe Analytics pour Power BI](https://powerbi.microsoft.com/en-us/documentation/powerbi-content-pack-adobe-analytics/).
