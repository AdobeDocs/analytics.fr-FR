# Ajout de requêtes dépendantes

{{legacy-arb}}

Une requête dépendante est une requête qui dépend d’une requête existante.

Pour les rapports dont les répartitions de corrélation sont activées dans le tableau de rapport, pour examiner de plus près un élément, cliquez avec le bouton droit dans la cellule, puis cliquez sur **[!UICONTROL Ajouter une requête dépendante]** > **[!UICONTROL Répartition]**.

>[!NOTE]
>
>Vous pouvez également ajouter une demande de répartition lors de la création de la demande d’origine. Reportez-vous à la section [Création d’une requête de données](/help/analyze/legacy-report-builder/data-requests/t-create-a-data-request.md).

>[!NOTE]
>
>Adobe Report Builder applique les dépendances de requête uniquement dans la même feuille de calcul, et non entre les feuilles de calcul. Ceci garantit la rapidité de l’exécution.
