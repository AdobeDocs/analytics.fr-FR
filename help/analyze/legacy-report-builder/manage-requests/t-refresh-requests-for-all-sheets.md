# Actualisation des requêtes de toutes les feuilles

{{legacy-arb}}

Vous pouvez actualiser les requêtes pour l’ensemble des feuilles ou pour plusieurs d’entre elles.

Pour actualiser toutes les requêtes, vous devez avoir accès à toutes les suites de rapports utilisées dans toutes les requêtes. Si vos informations d’identification ne vous donnent accès qu’à un sous-ensemble de toutes les suites de rapports utilisées, un message d’avertissement s’affiche lors de la connexion pour vous informer que certaines des requêtes ne peuvent pas être actualisées. Dans ce cas, si vous actualisez toutes les requêtes, celles auxquelles vous n’avez pas accès ne seront pas actualisées.

Lorsque vous actualisez plusieurs requêtes, le système rassemble et classe par priorité les requêtes selon qu’il s’agit d’une requête originale ou dépendante. Le système affiche des messages sur la progression de l’actualisation. Le processus d’actualisation peut prendre jusqu’à plusieurs minutes, selon la quantité de données que vous demandez.

Pour actualiser les demandes de toutes les feuilles

1. Cliquez sur **[!UICONTROL Gérer]**.
1. Activez l’option **[!UICONTROL Toutes les feuilles]**.
1. Cliquez sur **[!UICONTROL Actualiser]**.
1. Indiquez une référence de date, puis cliquez sur **[!UICONTROL OK]**.
