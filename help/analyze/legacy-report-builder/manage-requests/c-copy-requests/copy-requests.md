# Copie des requêtes - Aperçu

{{legacy-arb}}

Vous avez la possibilité de copier les cellules mises en correspondance avec plusieurs requêtes et de coller le contenu dans une zone sélectionnée vide de la feuille de calcul.

Une fois les cellules copiées, le Report Builder et Excel déterminent la zone nécessaire pour coller le nombre minimal de copies. Si la zone est suffisamment grande, le collage crée une copie de toutes les requêtes, où chaque requête collée a la même disposition spatiale et le même formatage que dans les requêtes d’origine.

Il s’agit de la propagation de la requête. Il s’agit d’une méthode simple et rapide pour créer un rapport long. Le Report Builder commence par propager les requêtes en les collant toutes dans les cellules de la région de collage ciblée, puis actualise les cellules en fonction des dates du rapport établies pour les requêtes.
