# Mise en page - Assistant Requête : Étape 2

{{legacy-arb}}

La fenêtre [!UICONTROL Assistant Requête : étape 2] vous permet de configurer la disposition du rapport, le formatage de l’en-tête et le filtrage des dimensions pour la sortie. Vous pouvez utiliser une disposition croisée dynamique pour configurer la requête, ce qui revient à créer un tableau croisé dynamique dans Excel.

Une autre solution consiste à sélectionner manuellement les cellules dans lesquelles vous souhaitez afficher les données du rapport en utilisant la [Disposition personnalisée](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md).
