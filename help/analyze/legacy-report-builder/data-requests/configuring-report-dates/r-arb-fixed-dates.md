# Définitions de l’Assistant Requête - Dates fixes

{{legacy-arb}}

Définitions des dates fixes dans le Report Builder.

| Champ | Définition |
|--- |--- |
| Cette suite de rapports | Rend la période enregistrée disponible uniquement pour la suite de rapports actuelle. |
| Toutes les suites de rapports | Rend la période enregistrée disponible pour toutes les suites de rapports accessibles à l’utilisateur connecté. |
| Saisir le nom du paramètre | Saisissez un nom pour la date enregistrée. Vous pouvez utiliser cette date comme modèle dans Report Builder. Ce nom s’affiche lorsque vous pouvez appliquer une date enregistrée. La modification du nom d’un modèle n’affecte pas la configuration de dates utilisée dans les demandes de données créées précédemment. |
