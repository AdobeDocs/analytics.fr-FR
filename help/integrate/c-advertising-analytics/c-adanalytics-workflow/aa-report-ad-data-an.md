# Rapport sur les données Advertising dans Adobe Analytics

Détails sur le modèle Analysis Workspace et sur la création de rapports dans Reports &amp; Analytics, ainsi que dans Report Builder.

>[!NOTE] Patientez 24 heures au moins avant que les données du moteur de recherche ne commencent à apparaître dans vos rapports Analytics. Notez également que le reporting Analytics ne renverra pas de données pour la granularité horaire, car les données AMO ne prennent pas en charge ce type de granularité.

## Analysis Workspace : Moteurs de recherche {#section_8173F42B2C784F41B9FD82CBB66F9ADF}

Ce modèle permet à quiconque mettant en œuvre une intégration de moteur de recherche d’accéder à de nombreuses données de moteur de recherche dans Analytics. Vous pouvez y accéder par **[!UICONTROL Workspace]** > **[!UICONTROL Templates]** > **[!UICONTROL Advertising]** > **[!UICONTROL Search Engines.]**

>[!NOTE]
>
>La catégorie de modèle Advertising est visible par tous les clients, y compris ceux qui n’ont implémenté aucun compte Advertising. Cependant, si vous essayez d&#39;ouvrir le modèle Moteurs de recherche pour un qui n&#39;a pas encore été configuré, un message d&#39;erreur vous expliquera que vous n&#39;avez pas encore configuré de compte de moteur de recherche. Dans ce cas, cliquez sur **[!UICONTROL Configure Now]**, ce qui vous mènera à l’écran Configuration [du compte](/help/integrate/c-advertising-analytics/c-adanalytics-workflow/aa-create-ad-account.md) publicitaire.
