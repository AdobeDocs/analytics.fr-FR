# Rapport sur la publicité dans Adobe Analytics

Détails sur le modèle Analysis Workspace et sur la création de rapports dans Reports &amp; Analytics, ainsi que dans Report Builder.

>[!NOTE]
>
>Vous devriez attendre d'attendre au moins 24 heures avant que les données du moteur de recherche ne soient renseignées dans vos rapports Analytics. Notez également que le reporting Analytics ne renverra pas de données pour la granularité horaire, car les données AMO ne prennent pas en charge ce type de granularité.

## Analysis Workspace: Search Engines {#section_8173F42B2C784F41B9FD82CBB66F9ADF}

Ce modèle permet à quiconque mettant en œuvre une intégration de moteur de recherche d’accéder à de nombreuses données de moteur de recherche dans Analytics. You can access it via **[!UICONTROL Workspace]** &gt; **[!UICONTROL Templates]** &gt; **[!UICONTROL Advertising]** &gt; **[!UICONTROL Search Engines.]**

>[!NOTE]
>
>La catégorie Modèle de publicité est visible pour tous les clients, même si vous n'avez implémenté aucun compte de publicité. Cependant, si vous tentez d’ouvrir le modèle Moteurs de recherche pour une entreprise dont la configuration n’a pas été effectuée, un message d’erreur vous indiquera que vous n’avez encore configuré aucun compte de moteur de recherche. Dans ce cas, cliquez sur **[!UICONTROL Configurer maintenant]**, vous serez alors dirigé vers l’écran [Configuration d’un compte Advertising](../../../integrate/c-advertising-analytics/c-adanalytics-workflow/aa-create-ad-account.md#concept_1958E8C15C334E8B9DC510EC8D5DCA7C).
