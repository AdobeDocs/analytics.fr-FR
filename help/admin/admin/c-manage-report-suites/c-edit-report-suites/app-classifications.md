# Classifications des applications

Les classifications mobiles sont activées une fois que vous avez activé l’une des fonctionnalités sous [Rapports d’application](app-reporting.md). Il fournit une interface identique à [Classifications des conversions](conversion-var-admin/conversion-classifications.md) ou [Classifications de trafic](c-traffic-variables/traffic-classifications.md), mais pour les dimensions de création de rapports d’application.

**[!UICONTROL Analytics]** > **[!UICONTROL Administration]** > **[!UICONTROL Suites de rapports]** > **[!UICONTROL Modifier les paramètres]** > **[!UICONTROL Gestion des applications]** > **[!UICONTROL Classifications des applications]**

Adobe recommande plutôt d’utiliser [Jeux de classifications](/help/components/classifications/sets/overview.md) pour toutes les nouvelles données de classification.
