# Rapports du Report Builder

Gérez les licences attribuées aux utilisateurs du Report Builder.

Chaque licence de Report Builder attribuée à un utilisateur correspond automatiquement à dix tâches planifiées. Ce nombre de tâches peut être ajusté et réaffecté. Les tâches planifiées ne peuvent pas dépasser le nombre de tâches disponibles pour chaque utilisateur. Aucune tâche planifiée n’est exécutée.

Vous pouvez accorder un accès au Report Builder à n’importe quel utilisateur [!DNL Adobe Analytics] en gérant les profils de produit dans [Adobe Admin Console](/help/admin/admin-console/home.md).
