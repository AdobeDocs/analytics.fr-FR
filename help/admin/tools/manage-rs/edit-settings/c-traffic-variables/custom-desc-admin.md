# Descriptions des rapports personnalisés

Vous pouvez fournir des descriptions de rapports personnalisées pour les eVars, les props, les classifications et les événements. Les descriptions sont visibles par tous les utilisateurs finaux.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Suites de rapports]** > **[!UICONTROL Modifier les paramètres]** > **[!UICONTROL *variable *]**

>[!NOTE]
>
>L’ajout de descriptions est disponible pour la plupart des classifications traditionnelles, mais pas actuellement pour les classifications mobiles.

![](/help/admin/tools/assets/report_descriptions.png)

1. Fournissez une description du rapport. Le nombre maximal de caractères est de 255.
1. Cliquez sur **[!UICONTROL Enregistrer]**.

La description est visible à deux endroits :

* Dans la barre de titre de ce rapport :

  ![](/help/admin/tools/assets/report_description_2.png)

* Comme info-bulle lorsque vous pointez sur le rapport dans le volet de navigation de gauche :

  ![](/help/admin/tools/assets/report_description_3.png)
