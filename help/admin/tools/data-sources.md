# Sources de données

La fonction Sources de données vous permet d’importer des données dans Analytics à partir de sources hors ligne. Une fois importées, ces données peuvent être traitées de la même manière que les données collectées en mode natif.

Utilisez les sources de données pour :

* Intégrez les données hors ligne aux données collectées sur votre site web.
* Collectez des données à partir de serveurs web, de centres d’appels ou d’autres systèmes. Importez-le ensuite dans CX Enterprise pour afficher la conversion de toutes les avenues de votre organisation.
* Optimisez la gestion des enchères par mot-clé en fonction de l’exécution des prospects hors ligne via SearchCenter.

Lorsque la fonctionnalité Sources de données est activée sur une suite de rapports, un emplacement FTP est fourni afin que vous puissiez charger des fichiers à intégrer dans les rapports. Une fois le chargement effectué, les serveurs Adobe traitent automatiquement les données et les intègrent à votre suite de rapports.

Voir [Aide sur les sources de données](/help/import/data-sources/overview.md).
