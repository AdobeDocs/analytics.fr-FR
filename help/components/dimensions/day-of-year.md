# Jour de l&#39;année

La dimension &quot;Jour de l’année&quot; signale le jour numérique d’une année donnée comme valeur de dimension. Ce rapport est utile si vous souhaitez qu’un rapport soit ventilé par jour de l’année, mais que vous ne souhaitez pas qu’une date statique soit une valeur de dimension.

## Renseigner cette dimension avec des données

Cette dimension est prête à l’emploi pour toutes les implémentations. Si une suite de rapports contient des données, cette dimension fonctionne.

## Valeurs de dimension

Les valeurs de dimension comprennent les nombres `1` (1er janvier) à `366` (31 décembre de l’année bissextile), qui représentent le jour de l’année où l’accès a eu lieu.
