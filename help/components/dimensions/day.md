# Jour

La dimension &quot;Jour&quot; signale le jour où une mesure donnée s’est produite. La première valeur de dimension est le premier jour de la période et la dernière valeur de dimension est le dernier jour de la période. Cette dimension est essentielle pour les rapports de tendances, car elle vous permet d’afficher les mesures au fil du temps.

## Renseigner cette dimension avec des données

Cette dimension est prête à l’emploi pour toutes les implémentations. Si une suite de rapports contient des données, cette dimension fonctionne.

## Valeurs de dimension

Les valeurs de dimension comprennent la date d’un jour donné. Elle inclut le mois, le jour et l’année dans la valeur de dimension.
