# Taux de rebond

La mesure Taux de rebonds présente le rapport des visites qui contenait exactement un accès par rapport au nombre de visites sur cette page. Vous pouvez utiliser cette mesure pour identifier les valeurs de dimension qui présentent le taux de rebonds le plus élevé ou pour afficher un taux de rebonds total agrégé de votre site au fil du temps.

## Méthode de calcul de cette mesure

Cette mesure est calculée à l’aide de la formule [`Bounces`](bounces.md)` divided by `[`Entries`](entries.md).
