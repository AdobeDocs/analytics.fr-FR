# Importation navigateur et FTP

Après avoir copié les données appropriées dans le fichier de modèle de classification, vous pouvez les importer (télécharger) dans des serveurs de collecte de données Adobe.

>[!NOTE] Vous ne devez pas télécharger (à l’aide de la fonction d’exportation) vos données de classification existantes avant de charger de nouvelles données. Il vous suffit de télécharger un nouveau fichier contenant les nouvelles lignes de données. Cette méthode améliore les performances.

See [General File Structure](/help/components/c-classifications2/c-classifications-importer/c-saint-data-files.md) for important information about the structure and limitations of uploaded files.

Pour plus d’informations sur la résolution des problèmes, reportez-vous à la section [Problèmes de téléchargement de classifications courants](https://helpx.adobe.com/fr/analytics/kb/common-saint-upload-issues.html).
